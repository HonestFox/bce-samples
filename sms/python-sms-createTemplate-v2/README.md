# python-sms-createTemplate-v2

## 用途：

创建短信模板（接口版本：v2）。

**注意：仅适用于Python 2.7**

## 使用方法：

* 第一步：在文件`sms_client_conf.py`中配置AK/SK。
* 第二步：在文件`create_template.py`中填写要创建的短信模板的相关信息。
* 第三步：执行文件`create_template.py`。
