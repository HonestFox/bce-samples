# java-iothub-detachPrincipalPolicy

## 用途：

解除身份（principal）与策略（policy）的绑定关系。

## 使用方法：

* 第一步：在`App.java`中填写ak/sk，以及endpointName、principalName、policyName。
* 第二步：执行`App::main`方法。
