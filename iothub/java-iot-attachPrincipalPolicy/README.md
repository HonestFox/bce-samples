# java-iothub-attachPrincipalPolicy

## 用途：

把策略（policy）绑定到一个身份（principal）上。

## 使用方法：

* 第一步：在`App.java`中填写ak/sk，以及endpointName、principalName、policyName。
* 第二步：执行`App::main`方法。
