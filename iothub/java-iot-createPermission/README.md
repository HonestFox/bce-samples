# java-iothub-createPermission

## 用途：

创建主题（topic）并配置相应的权限（PUB、SUB）。

## 使用方法：

* 第一步：在`App.java`中填写ak/sk以及endpointName、policyName、topic以及对应的权限。
* 第二步：执行`App::main`方法。
