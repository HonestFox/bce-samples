# java-iothub-attachThingPrincipal

## 用途：

把身份（principal）绑定到一个设备（thing）上。

## 使用方法：

* 第一步：在`App.java`中填写ak/sk，以及endpointName、thingName、principalName。
* 第二步：执行`App::main`方法。
