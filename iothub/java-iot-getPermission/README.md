# java-iot-getPermission

## 用途：

获取指定的topic信息。

## 使用方法：

* 第一步：在`App.java`中配置AK/SK，以及endpointName、permissionUuid。
* 第二步：执行`App::main`方法。

**注意：permissionUuid 可以通过 [java-iot-getPermissions](../java-iot-getPermissions) 查询。**
