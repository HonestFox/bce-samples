# IoT samples

项目命名规则：

* `java-iot-pub-ssl`
    * java：编程语言为Java。
    * iot：云服务为IoT。
    * pub：发布（publish）消息。
    * ssl：使用SSL连接方式。
* `java-iot-sub-ssl`
    * java：编程语言为Java。
    * iot：云服务为IoT。
    * sub：订阅（subscribe）消息。
    * ssl：使用SSL连接方式。
